#Java MYSQL Console App
A simple Java console application for managing bookstore records using MySQL and JDBC.

#Features:
-Add books
-View books
-Update books
-Delete books
-MySQL database connectivity

#Technologies used:
-Java
-MySQL
-JDBC

#To Run:
javac BookstoreApp.java
java BookstoreApp

#Database Setup
create a MySQL database, create a table, configure and connect it in the Java file using your username and password.

#Pre-requistics:
before running the project, make sure you have:
-Java JDK installed
-MYSQL server installed
-MySQL JDBC Driver connector
-Any Java IDE or terminal
